# Django-vue-smolapp

## Project setup

### backend setup
```
pip install requirements.txt
cd vue_project
python manage.py runserver 0.0.0.0:5000
```

### frontend setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
